# Security Policy

## Scope

Aye Chef is designed for trusted home networks. It does not include built-in user authentication.

## Reporting a Vulnerability

If you discover a security issue, please report it **privately** using [GitHub Security Advisories](../../security/advisories/new).

Please do **not** open a public issue for security vulnerabilities.

## What to Report

- Credential leakage (tokens/keys exposed in logs, responses, or UI)
- Remote code execution or command injection
- Cross-site scripting (XSS) in the web panel
- Any issue that could affect users on a shared network
